===

This modpack is an attempt to recreate Life in the Woods Renaissance for Minecraft 1.12.2 (and higher?).

---

Included are the following mods. If there is an update and I missed it, feel free to open an issue so that I can update the mod.

-   [CurseForge](https://minecraft.curseforge.com/projects/animania)
